# Python Excel Data Cleaning Tool

A Python automation tool that cleans and prepares Excel data by removing empty rows and duplicate records. Designed to help businesses and individuals work with accurate and organized datasets.

**What this tool does:**
This script reads an Excel file, removes unnecessary or messy data, and saves a cleaned version automatically. It recuces manual work and improves data quality.

**Features:**
* Teads Excel files automatically
* Removes completely empty rows
* Removes duplicate records
* Option to remove rows with missing important data (e.g., Email)
* Saves a new cleaned Excel file

**Technologies Used:**
Python, pandas, openpyxl

**How it works:**
1. The script loads an Excel file using pandas
2. It scans for empty rows and removes them
3. It detects duplicate entries and removes them
4. A cleaned Excel file is generated and saved

**Example Use Cases:**

* Cleaning customer databases
* Preparing reports
* Removing duplicate entries in business records
* Fixing messy exported system data

**Why This Project Matters:**
This project demonstrates real-world data cleaning skills. Businesses frequently deal with messy spreadsheets, and this tool shows how Python can automate and simplify data preparation tasks.
